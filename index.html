Spotibine

Using shuffle mode on a playlist is one of the most popular ways to listen to music on Spotify. However, many medleys and others song combinations featured on albums are incompatible on shuffle mode. Spotify cannot ensure that compositions including Brain Damage/Eclipse, We will rock you/We are the champions, and the Abbey Road medley are played together.

Spotibine is a web-based application that allows the user to create custom song sequences on Spotify Web. Users can play medleys on playlist in shuffle mode without having to manually queue them.

Chip-8 Emulator

According to Wikipedia:
CHIP-8 is an interpreted programming language, developed by Joseph Weisbecker on his 1802 microprocessor. It was initially used on the COSMAC VIP and Telmac 1800, which were 8-bit microcomputers made in the mid-1970s.
Many simple games, such as Space Invaders and Pong, can be programmed into Chip-8 files. This program is an emulator for Chip-8 files to be runnable with C++. Compatible with keyboard controls.
This emulator has been tested and passed by this test suite. To run the program, store the Chip8 file in the same directory as the CMake program, and run the following command in terminal:
./chip8 {scale} {delay} {ROM name}

Octical

Have you ever entered a public place (e.g. Gyms, Libraries, etc.) and become disappointed that it was crowded? Octical is an application using React Native that could tell you beforehand the density of the space before you travel.

Octical currently enables its users to upload pictures, which will activate a computer vision program that will compute and update the crowdiness of the space for other users to see. Still in-progress, this is a group project involving contributions from six University of Waterloo students.
